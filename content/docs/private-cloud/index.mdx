私有云文档用于说明如何在企业自有环境中部署、激活、运维和治理 Sealos。它是公有云主线文档之外的扩展专区，主要服务于交付、管理员和运维角色。

## 适合谁看

- 需要离线部署或内网交付的团队
- 关注数据本地化与合规要求的组织
- 需要平台管理员、运维和交付角色协同推进的项目
