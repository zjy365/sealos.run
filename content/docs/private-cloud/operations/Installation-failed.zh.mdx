建议按下面顺序排查：

1. 检查日志：`tail -f /root/.sealos/cloud/install.log`
2. 验证配置：确认 `sealos.env` 和 `alias.yaml` 正确
3. 检查资源：确认节点 CPU、内存、磁盘充足
4. 网络测试：确认节点间网络互通，SSH 可达
5. 回看部署准备、许可证和故障排查文档
