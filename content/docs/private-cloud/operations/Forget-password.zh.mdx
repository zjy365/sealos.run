获取管理员密码：

```bash
kubectl get secret admin-password -n sealos-system -o jsonpath='{.data.password}' | base64 -d
```

获取 Registry 密码：

```bash
kubectl get configmap registry-config -n sealos-system -o jsonpath='{.data.ADMIN_PASSWORD}'
```
